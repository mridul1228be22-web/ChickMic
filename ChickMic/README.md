# Chickmic Fashion

## Run the project

From this folder (`ChickMic`):

```bash
npm run dev
```

Or from the parent `Project_FInal` folder:

```bash
npm run dev
```

Open http://localhost:5173

Both the frontend (port 5173) and API (port 3001) must be running. You should see `[api] Connected to MongoDB` in the terminal.

## If login fails

1. Make sure `.env` exists in this `ChickMic` folder (not only in the parent folder).
2. Restart with `npm run dev` after any `.env` change.
